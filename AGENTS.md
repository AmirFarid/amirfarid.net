# Agent Notes

This is Amir M. Farid's personal academic website.

## Design

- Modern professional minimal
- Academic, readable, technical
- Heavy animation only inside Learning posts when it improves explanation

## Stack

- Astro
- TypeScript
- Markdown/MDX
- Plain CSS
- React only for future interactive learning components
- React Three Fiber only for future 3D teaching posts

## Rules

- Do not add large dependencies without justification.
- Do not use Tailwind unless explicitly requested.
- Do not invent publications, credentials, or links.
- Do not modify publication/CV claims without confirmation.
- Do not directly edit publication entries unless the user explicitly requests that specific publication change.
- Do not directly edit `public/cv.pdf`, CV claims, or CV-related text unless the user explicitly requests it.
- Do not change research claims, biography claims, author lists, publication status, venue, DOI, affiliation, or credential wording without explicit confirmation.
- Do not change deployment settings, hosting configuration, build output mode, CI secrets, or domain settings unless explicitly requested.
- Keep normal pages static and fast.
- Keep components small and reusable.
- Prioritize accessibility and performance.
- Use the content templates and `npm run new:*` scripts when adding new collection entries.
- Keep generated metadata such as `public/version.json` factual and machine-generated.
